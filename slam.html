PROJET SLAM

Ce projet porte sur la création d’un site web autour d’un thème en particulier, pour mon projet ce sera les différents MEME. Le projet consiste donc à créer un site contenant des formulaires d’inscription au site afin de pouvoir acheter et/ou poster des commentaires sur les différents MEME listés.

Pour la première étape j’ai du définir mes différentes tables quii constituent ma base de donnée. Par la suite j’ai du remplir certaines d’entre elles directement en ligne de commande sur PHPMyadmin (SQL).

Pour la seconde étape j’ai du remplir le contenu de mon site, c’est à dire la page d’accueil, la liste contenant les MEME en forme de grid et créer des formulaires d’inscriptions, de modification et de suppression de compte, mais aussi un formulaire qui permet à l’utilisateur inscrit de rentrer son avis sur l’un des MEME, tout celà avec du HTML & CSS codé en local sur VSCode.

Troisième étape : l’insertion des données rentrées par l’utilisateur voulant s’inscrire – modifier ou supprimer son compte, mais aussi l’insertion de son avis. Pour celà j’ai utilisé du PHP et du SQL pour injecter ce que l’utilisateur inscrit directement dans la base de données.

Pour terminer voici le rendu des formulaires sur la page HTML. Lorsque vous choisissez l’action que vous voulez, le formulaire qui sera demandé sera afficher.
